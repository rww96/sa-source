Save your return - Privacy guidance

You have chosen to view, print or save your return or Attachments. Your browser automatically stores a copy of the file in the computer's Temporary Internet Files "cache".

We suggest that if you are not using your own computer, or have concerns about the security of the computer, you delete this file once you have viewed or printed it.

The most common way of doing this is to select 'Tools' from the browser menu, then select 'Internet options' followed by 'Delete Files'.

If these instructions are not applicable to the browser you are using please refer to the 'Help' section in your browser, which should give instructions on how to do this.
